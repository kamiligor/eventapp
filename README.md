# Event Registration API

This is a sample API that allows you to save events in mongoDB.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app.<br />

### `npm test`

Launches the test runner.<br />


## Endpoints

/status<br />
/create<br />
/:id<br />
/:id/update<br />
/:id/delete
